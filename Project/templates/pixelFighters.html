Pixel Fighters

2D 1v1 single-screen fighting game

Pixel Fighters is a 2D fighting game developed using Unity. The game features pixelated characters battling in single-screen, 1v1 matches. It was developed during the end of 2023 as a personal project to experiment with game development and showcase Unity's capabilities for creating 2D games.
